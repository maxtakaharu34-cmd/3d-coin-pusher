# 3D Coin Pusher (3Dコインプッシャー)

ブラウザだけで動く 3D メダルゲーム(コインプッシャー)。Three.js による 3D 描画と cannon-es による物理演算で、メダルの落下・積み重なり・押し出しをリアルに再現しています。

## 遊び方

- **クリック / スペース** … メダル投入
- **マウス左右** … 投入位置を調整
- 手前**中央(緑ゾーン)**に落とすと獲得、**両端(赤)**は紛失
- 画面上部の HUD … 残メダル / 獲得 / 盤面のメダル数
- 獲得したメダルは残メダルに還元され、遊び続けられます

## 起動方法

ES Module + CDN を使用しているため、ローカルサーバー経由で開いてください。

```bash
cd 3d-coin-pusher
npx serve .
```

表示された URL(例 `http://localhost:3000`)をブラウザで開きます。

## 技術構成

- [Three.js](https://threejs.org/) — 3D 描画・ライティング・シャドウ
- [cannon-es](https://github.com/pmndrs/cannon-es) — 剛体物理エンジン(衝突・摩擦・キネマティックなプッシャー)
- ビルド不要・依存パッケージなし(CDN から import map で読み込み)
